# Dominio de la aplicación

## Video

Representa un video disponible para descargar.

Información principal:

- Título
- Autor
- Duración
- Plataforma
- Miniatura
- URL
- Formatos disponibles

## VideoFormat

Representa una opción de descarga.

Información:

- Calidad
- Extensión
- Tamaño
- Tipo (audio o video)

## VideoService

Responsable de obtener la información de un video y convertir los datos de yt-dlp al modelo de dominio de la aplicación.
